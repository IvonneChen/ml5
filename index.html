Why

The main idea of this project is to further reduce the barriers between lower level machine learning and creative outputs using javascript. p5ML provides two main functionalities:

A wrapper around deeplearn.js

Providing a simplier interface, that makes it easier to work with GPU accelerated machine learning in javascript.

Custom methods

A set of ready-to-use machine learning utilities.
